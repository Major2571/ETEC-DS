1. O que é um programa de computador?
Conjunto de instruções que descrevem uma tarefa a ser realizada por um computador.

2. O que é um sistema?
É um Conjunto integrado de componentes regularmente inter-relacionados e interdependentes criados para realizar um objetivo definido, com relações definidas e mantidas entre seus componentes e cuja produção e operação como um todo é melhor que a simples soma de seus componentes.

6. Como funciona a linguagem Java?
Java é uma linguagem de programação orientada a objetos, baseada em classes, que permite o desenvolvimento de aplicações pelos profissionais de TI através da escrita do código uma única vez.

7. O que é a máquina virtual?
Basicamente, máquina virtual é um programa que permite rodar um sistema operacional dentro de outro, sendo capaz de executar sistemas operacionais e aplicativos como se fosse uma máquina física, podendo, inclusive, rodar sistemas operacionais que normalmente não seriam compatíveis com sua arquitetura.

8. O que é JVM?
JVM, ou, Máquina virtual Java, é um programa interpreta o código em Java e permite rodá-lo em qualquer plataforma.

9. O que é JRE?
JRE, ou, Java Runtime Environment, é o ambiente de execução Java que deve ser instalado na máquina para rodar aplicativos Java em seu computador.

10. O que é JDK?
JDK, ou, Java Development Kit, é um pacote que inclui tudo o que é necessário para escrever aplicações e também deve ser instalado na sua máquina.
